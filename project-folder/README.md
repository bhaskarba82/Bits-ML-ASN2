# Machine Learning Classification Project

## Problem Statement
Build and compare six ML classification models to predict whether income exceeds $50K using Adult Income dataset.

## Dataset Description
- Instances: 48,842
- Features: 14
- Type: Binary Classification

## Models Used
1. Logistic Regression
2. Decision Tree
3. KNN
4. Naive Bayes
5. Random Forest
6. XGBoost

## Evaluation Metrics
Accuracy, AUC, Precision, Recall, F1 Score, MCC

## Deployment
Deployed using Streamlit Community Cloud.
